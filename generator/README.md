# Code generator plugin

## Instalacija

Potrebno podesiti `MAGICDRAW_HOME` ili `MAGICDRAW_HOME_UNIX` putanju u `build.properties` u zavisnosti od sistema i uvezati sve potrebne biblioteke.

Potrebna Java verzija >= 8
